# tyrion-itunes

## description
responsive browser of iTunes search API

using next.js as an application architecture framework

## requirements
node 6.9.x
npm 3.10.x

## getting started
```
npm install
npm run dev
```

## deploy
see here: https://tyrion-itunes.herokuapp.com/podcast?search=javascript
```
npm install
npm run build
npm start
```
